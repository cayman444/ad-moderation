# Веб-приложение для модерации объявлений

Система управления объявлениями для внутренней модерации контента. Позволяет модерировать объявления, отслеживать статистику и принимать решения по публикациям.

### **Запуск приложения**

```bash
# Клонирование репозитория
git clone https://github.com/cayman444/ad-moderation.git

# Установка и запуск сервера (терминал 1)
cd tech-int3-server
npm install
npm run dev

# Установка и запуск клиента (терминал 2)
cd client
npm install
npm run dev
```

### **Стек**

`React`, `TypeScript`, `Redux`, `RTK Query`, `React Router Dom`, `Ant Design`, `Tailwind CSS`, `Vite`

### **Функциональность**

- Просмотр списка объявлений с фильтрацией по статусу и категориям
- Детальный просмотр объявлений с историей модерации
- Принятие решений: одобрение, отклонение, возврат на доработку
- Указание причин отклонения с комментариями
- Статистика модерации
- Пагинация и сортировка
- Поиск по объявлениям с debounce
- Адаптивный интерфейс

### **Обоснование выбора технологий**

Для стилизации в проекте используется Tailwind CSS, чтобы сократить время написания стилей, так же библиотека Ant Design, которая предоставляет готовые ui компоненты. В приложении используется RTK Query, это позволило сократить код для работы с API и кешировать запросы. Проект собран на Vite, это так же значительно ускорило процесс разработки.
